# The Contemporary World - Educational Website

An interactive educational website for GEC-9: The Contemporary World course at Camarines Norte State College.

## 📚 Features

### Subject Modules
- **What is International Law** - Introduction to principles and sources of international law
- **A World of Regions** - Exploring regional blocs and their significance
- **Structure of Globalization** - Key economic and political architecture
- **Understanding Globalization** - Core concepts and historical dimensions
- **Legal Basis of International Relations** - Foundation through formal agreements

### Interactive Quiz System
- 🎲 **Random Quiz Challenge** - Questions from all topics
- 📝 **Topic-Specific Quizzes** - 10 questions per module
- 🎯 **Multiple Question Types** - Multiple choice and identification
- 📊 **Quiz History** - Track your progress and scores
- ✅ **Answer Review** - Review correct and incorrect answers

### Dark Mode
- 🌙 Toggle between light and dark themes
- 💾 Preference saved across sessions
- 🎨 Consistent styling across all pages

### About Us
- 👥 Team member profiles
- 📧 Contact information

## 🚀 Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/contemporary-world.git
```

2. Open `explore.html` in your web browser to start
